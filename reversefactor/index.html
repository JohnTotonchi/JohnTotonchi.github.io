Writing a Polynomial Function in Standard Form Given Zeros

Step-by-step Guide:

1. Write each zero as a factor, setting each factor equal to zero in the form of \((x - \text{zero})\).

2. Multiply all the factors together, resulting in the polynomial in standard form.

Example 1: Zeros are 3, -2, 1

We first write each zero as a factor: \((x - 3)\), \((x + 2)\), and \((x - 1)\).

Then, we multiply all the factors together to get the polynomial in standard form: \(f(x) = (x - 3)(x + 2)(x - 1)\).

Example 2: Zeros are -5, 4

We first write each zero as a factor: \((x + 5)\) and \((x - 4)\).

Then, we multiply all the factors together to get the polynomial in standard form: \(f(x) = (x + 5)(x - 4)\).

Example 3: Zeros are 1, -1, 0

We first write each zero as a factor: \((x - 1)\), \((x + 1)\), and \(x\).

Then, we multiply all the factors together to get the polynomial in standard form: \(f(x) = (x - 1)(x + 1)(x)\).
