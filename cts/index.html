Solving by Completing the Square

Step-by-step Guide:

1. Write the equation in the form \(ax^2 + bx + c = 0\).

2. If a ≠ 1, divide the entire equation by a to make the coefficient of \(x^2\) equal to 1.

3. Subtract c from both sides.

4. Take half of the coefficient of x, square it, and add it to both sides of the equation.

5. Factor the left side of the equation.

6. Solve for x via the square root method.

Example 1: \(x^2 - 6x + 8 = 0\)

Here, a = 1, b = -6, and c = 8.

Subtract 8 from both sides to get: \(x^2 - 6x = -8\).

Add \((b/2)^2 = (6/2)^2 = 9\) to both sides: \(x^2 - 6x + 9 = 1\).

Factor the left side to: \((x-3)^2 = 1\).

Last, solve for x: \(x - 3 = \pm\sqrt{1}\). This gives \(x = 3 \pm 1\), so the solutions are \(x = 2,4\).

Example 2: \(2x^2 - 8x + 7 = 0\)

First, divide the equation by 2 to simplify: \(x^2 - 4x + 3.5 = 0\).

Subtract 3.5 from both sides: \(x^2 - 4x = -3.5\).

Add \((b/2)^2 = (4/2)^2 = 4\) to both sides: \(x^2 - 4x + 4 = 0.5\).

Factor the left side: \((x-2)^2 = 0.5\).

Solve for x: \(x - 2 = \pm\sqrt{0.5}\). The solutions are \(x = 2 \pm \sqrt{0.5}\).
