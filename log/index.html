Logarithms: Solving, Expanding, Simplifying, and Converting from Exponential Forms

Understanding Logarithms

\(\log_b a = x\) means \(b^x = a\).

Solving Logarithmic Equations

To solve \(\log_b a = x\), remember that it is equivalent to \(b^x = a\).

Example: Solve \(\log_2 8 = x\)

Solution: Since \(2^3 = 8\), \(x = 3\).

Logarithmic Properties

The three properties of logarithms are:

1. Product Rule: \(\log_b (xy) = \log_b x + \log_b y\)

2. Quotient Rule: \(\log_b (x/y) = \log_b x - \log_b y\)

3. Power Rule: \(\log_b (x^n) = n \log_b x\)

Expanding Logarithms

Use the properties of logarithms to expand logarithmic expressions. For example, expand \(\log_b (x^2y)\).

Solution: Using the properties of logs, we get \(\log_b (x^2y) = 2\log_b x + \log_b y\).

Simplifying Logarithms

Simplify logarithmic expressions by using the properties of logs. For example, simplify \(2\log_b x + \log_b y\).

Solution: By simplifying, we get \(2\log_b x + \log_b y = \log_b (x^2y)\).

Converting from Exponential Forms

To convert \(b^x = a\) into a logarithmic form: \(\log_b a = x\).

Example: Convert \(2^3 = 8\) into a logarithmic form.

Solution: The logarithmic form is \(\log_2 8 = 3\).

Change of Base Formula

The change of base formula allows you to change the base of a logarithm. It is defined as \(\log_b a = \frac{\log_k a}{\log_k b}\) for any positive base \(k\) that's not equal to \(1\).

This is particularly useful if your calculator doesn't support logarithms to any base other than \(10\) or \(e\).

How to Use Your Calculator for Logarithms with Any Base

To calculate \(\log_b a\) on a calculator that only has \(\log\) (log base 10) and \(\ln\) (log base \(e\)), follow these steps:

  • For a base \(10\) logarithm, directly input \(\log a\) into your calculator.
  • For a base \(e\) logarithm, directly input \(\ln a\) into your calculator.
  • For logarithms of any other base, use the formula \(\log_b a = \frac \(\log a / \log b\) or \(\ln a / \ln b\) and input that into your calculator. You can also use the base \(10\) logarithm button (usually labeled as "LOG") or the natural logarithm button (usually labeled as "LN").

Example: Calculate \(\log_2 16\)

  • Using base \(10\): Enter \(\log 16 / \log 2\) into your calculator.
  • Using base \(e\): Enter \(\ln 16 / \ln 2\) into your calculator.

The result will be \(4\), since \(2^4 = 16\).

Solving Exponential Equations with a Common Base

To solve an exponential equation where both sides can be expressed with a common base, you equate the exponents of those bases and solve for the unknown.

Example: Solve \(2^{x+1} = 2^3\)

Solution: Since both sides of the equation have a common base of \(2\), we can set the exponents equal to each other:

\(x + 1 = 3\)

Subtracting \(1\) from both sides gives \(x = 2\).

Steps to Solve Exponential Equations with a Common Base

  • Ensure that both sides of the equation are expressed as powers of the same base.
  • Equate the exponents of these bases.
  • Solve the resulting equation for the unknown.

Tip: If the equation cannot be directly rewritten to have a common base, try using logarithms to solve it.
